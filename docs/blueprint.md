# **App Name**: CampusConnect

## Core Features:

- User Authentication: User authentication with student, professor, and admin roles.
- Personalized Dashboard: Personalized home screen with event suggestions based on user preferences and school affiliation.
- Event Management: Event creation and listing with filtering, sorting, and map/list views.
- Real-time Messaging: Private and group chat functionality for communication.
- User Network: Follow system for connecting with students who share interests and creating a small social network.
- AI Event Recommendations: AI-powered event recommendations using a tool that analyzes student profiles and event metadata.
- Event Notifications: Notifications and calendar integration to keep users informed about upcoming events.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a sense of calm and productivity, reflecting the 'coworking' aspect. In HSL: H200, S47, L79.
- Background color: Very light blue (#F0F8FF), a barely-there tint of the primary. In HSL: H208, S28, L97.
- Accent color: Lavender (#E6E6FA) to evoke creativity and a social atmosphere, complementary to the light blue. In HSL: H240, S40, L94.
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif.
- Clean and modern icons representing event types and categories.
- Intuitive card-based design for event listings and user profiles.
- Subtle animations and transitions to enhance user engagement and navigation.
